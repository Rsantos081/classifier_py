# Sistema de Classificação de Chuvas

Projeto desenvolvido em Python para classificar níveis de chuva, registrar usuários e armazenar dados automaticamente em arquivos e pastas.

## Funcionalidades

- Registro de usuários
- Criação automática de pastas
- Gravação de dados em arquivos `.txt`
- Classificação do nível de chuva
- Cálculo do total de chuva
- Exibição de relatórios no terminal

## Classificação de Chuva

- Até 20 mm → Baixo (Verde)
- De 21 a 30 mm → Médio (Amarelo)
- Acima de 30 mm → Alto (Vermelho)

## Estrutura de Pastas
